# Windows 11 Portfolio

A personal portfolio reimagined as a Windows 11 desktop — running entirely in the browser. Instead of a traditional scrollable page, visitors interact with a full desktop environment: opening windows, using the taskbar, browsing the start menu, and even running terminal commands.

---

## What it looks like

The desktop greets visitors with a Windows 11-style wallpaper and app icons. Clicking any icon opens a floating, draggable window — just like a real OS.

## Apps

| App          | Description                                                                   |
| ------------ | ----------------------------------------------------------------------------- |
| **About**    | Profile photo, bio, role, and social links                                    |
| **Projects** | Project cards with thumbnails, descriptions, and tech tags                    |
| **Skills**   | Categorized skill badges — languages, frameworks, tools, design               |
| **Contact**  | Contact form + email info                                                     |
| **Terminal** | Interactive terminal with commands like `help`, `about`, `projects`, `skills` |

---

## Interactions

- **Draggable windows** — click and drag any window by its title bar
- **Minimize / Maximize / Close** — full window controls
- **Taskbar** — shows open apps, click to switch or minimize
- **Start Menu** — pinned apps, recommended links, user profile
- **Shutdown animation** — click the power button for a full Windows shutdown sequence

## Tech Stack

- **React 19** — UI

- **Framer Motion** — window animations
- **Lucide React** — icons
